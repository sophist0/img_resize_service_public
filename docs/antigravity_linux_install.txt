I have found antigravity ide is a pain to install on linux. Last time I did it I did not document how I did it. So hopefully this will do that.

---------------------------------------------------------
Install Antigravity 2.0
---------------------------------------------------------

This looks similiar to what I did before: https://www.reddit.com/r/google_antigravity/comments/1tksqle/installing_antigravity_20_on_ubuntu/

---------------------------------------------------------
Install Antigravity IDE
---------------------------------------------------------

Download the IDE here: https://antigravity.google/download#antigravity-ide

Careful this page includes Antigravity, the CLI, the IDE, and the SDK.

Uncompress the tar:

	tar -xvf Antigravity\ IDE.tar.gz

This just unzips the IDE it does not install anything. To install it

	sudo mv Antigravity\ IDE /opt/antigravity-ide

Add a Destop Entry for the IDE: ~/.local/share/applications/antigravity-ide.desktop

[Desktop Entry]
Name=Antigravity IDE
Comment=Google Antigravity Code Editor
Exec=/opt/antigravity-ide/antigravity-ide %U
Terminal=false
Type=Application
Icon=development
Categories=Development;IDE;TextEditor;
StartupNotify=true
SartupWMClass=Antigravity IDE
MimeType=x-scheme-handler/antigravity-ide;

Then run:

	chmod +x ~/.local/share/applications/antigravity.desktop

	update-desktop-database ~/.local/share/applications

Also have to update the owner of antigravity-ide

	sudo chown root:root /opt/antigravity-ide/chrome-sandbox

	sudo chmod 4755 /opt/antigravity-ide/chrome-sandbox
